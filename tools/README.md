# Task Nexus 激活码生成指南

## 您（卖家）的操作流程

### 步骤 1：获取用户机器码
用户启动软件后会看到激活对话框，上面显示他们的**机器码**（16位大写字母数字），如：
```
A1B2C3D4E5F6G7H8
```
用户复制后发给您。

---

### 步骤 2：生成激活码

打开终端（命令行），进入项目的 `tools` 目录，运行：

```bash
cd /Users/lv/Workspace/Task\ NeXus/tools
node license-generator.js <用户机器码>
```

**示例**：
```bash
node license-generator.js A1B2C3D4E5F6G7H8
```

**输出**：
```
╔══════════════════════════════════════════════════╗
║           许可证生成成功！                       ║
╠══════════════════════════════════════════════════╣
║  机器码:   A1B2C3D4E5F6G7H8                      ║
║  激活码:   F9404114B0B11C13                      ║
╚══════════════════════════════════════════════════╝
```

---

### 步骤 3：发送激活码给用户
将生成的**激活码**（16位）发给用户，用户在软件激活对话框中输入后点击"激活软件"即可。

---

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 用户换电脑了 | 需要用新机器码重新生成激活码 |
| 激活码无效 | 检查机器码是否复制完整（16位） |
| 找不到 node 命令 | 安装 Node.js |

---

## 安全提醒

> ⚠️ `license-generator.js` 文件和密钥 `LICENSE_SECRET` **请勿泄露**！
> 
> 如需更换密钥，同时修改：
> - `electron/main.js` 第 133 行
